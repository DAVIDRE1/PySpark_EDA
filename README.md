# PySpark_EDA
USA Flight delay EDA using PySpark


Goal: Exploratory data analysis on USA Flight delay 

Data: find data in https://www.transtats.bts.gov/Fields.asp?gnoyr_VQ=FGJ

What I used: PySpark and matplotlib
